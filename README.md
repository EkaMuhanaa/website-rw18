# 🏘️ Website Company Profile RW 18

Website Company Profile RW 18 merupakan sistem informasi berbasis web yang dirancang untuk membantu pengurus RW dalam mengelola informasi dan administrasi warga secara digital, sekaligus menjadi media publikasi bagi masyarakat.

---

## ✨ Fitur Utama

### 🌐 Website Publik
- Beranda
- Profil RW
- Struktur Organisasi
- Data Aset RW
- Agenda Kegiatan
- Galeri Dokumentasi
- Informasi Kontak

### 🔐 Dashboard Admin
- Login Admin
- Dashboard
- Kelola Data Warga
- Kelola Data Aset
- Kelola Agenda
- Kelola Galeri
- Manajemen User
- Edit Profil
- Soft Delete Data Warga

---

## 🛠 Tech Stack

| Teknologi | Digunakan |

## 🚀 Cara Instalasi

```bash
git clone https://github.com/EkaMuhanaa/website-rw18.git

cd website-rw18

composer install

npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

npm run dev

php artisan serve
```
